## ⚠️ 現アーキテクチャの限界 (正直な評価)

### L1. Mos 公式サイトの SPA 制約
- `www.mos.jp/shop/detail/?shop_cd=XXXXX` は JavaScript 動的 rendering のため
  httpx + BeautifulSoup での scraping では **operator 情報が HTML に存在しない**
- 結果として per_store 抽出では 363 店舗全てに「株式会社モスフードサービス
  (本部)」のみしか紐付けできない

### L2. Google Places API の registration-level 粒度
- 各店舗の `official_url` は全店一律 `www.mos.jp/shop/detail/...` (本部 URL)
- Places 側のデータに operator (FC 加盟企業) 情報を持たない
- **Phase 18 URL ドメイン二次 filter** で 6 件の BK 誤登録は除外できたが、
  FC 識別は別問題

### L3. Top-down search_by_operator の限界
- registry 登録 FC 10 社を `places_client.search_by_operator(name, "東京都")` で
  検索しても、東京都には本社を持たない地方 FC ばかりで place_id 突合は 0 件
- audit CSV: 東京都 Mos 192 店 × 登録 13 社 = **coverage 0-40%** 止まり

### L4. 東京都内 polygon filter の漏れ
- bake 384 件中、192 件が越境 (埼玉/神奈川/千葉/山梨)
- `menu.ResolvePolygon` の 東京都 bbox + post-filter で近県店舗の
  誤取り込みを完全排除できていない。Phase 21 候補 (polygon tightening)

> 「東京都内のモスバーガー店舗について全てを洗い出し、これら一つ一つに対しての
> 事業会社を確かめ、その事業会社についてのリストをしっかりと適切に証拠付きで」

**できたこと**:
- 東京都 Mos 192 店舗の place_id / 住所 / 緯度経度 / 公式 URL を CSV 化
  → `var/mos-tokyo-fullrun/final/tokyo-stores.csv`
- 全国で Mos を運営している 13 社 (+ 本部 1 社) を gBizINFO 法人番号つきで特定
  → `var/mos-tokyo-fullrun/final/mos-operators-evidence.csv`

**構造的にできなかったこと**:
- 個別 192 店舗 ↔ 個別事業会社 の**1:1 紐付け**

- 唯一の例外は「モスストアカンパニーが東京都の大部分を運営している」という
  定性的結論 (子会社 + 東京都本社 + 全国 209 店の重みから)

## 📈 次のアクション (Phase 21 候補)

個別店舗 ↔ 個別事業会社の 1:1 紐付けを実現するには:

1. **Mos 側の公開 FC マップ入手** — IR 資料 (有価証券報告書) の重要契約先や
   「モスバーガー共栄会」全国 20 支部名簿の scrape (各支部 FC 加盟企業一覧)
2. **browser-use 導入で SPA rendering** — 既存の `_default_browser_agent` を
   OperatorSpider に注入、Mos shop_cd ページから FC 名を JS 待機 fetch
3. **FC 各社公式の店舗一覧 scrape** — モスストアカンパニー/ヴィアン/大和フーヅ
   の自社サイトから「営業中の店舗一覧」を抜き住所で突合
4. **Web search agent による東京都ローカル FC 調査** — 本レポートでは時間制限で
   完了せず。追加調査で小規模個人 FC (1-3 店規模) を発見できる可能性あり
5. **gBizINFO 法人名全件横断 match** — 国税庁 API で「モスバーガー」「モス」
   を含む法人を抽出し、gBizINFO 登録の事業目的と突き合わせ

**現時点で PI-ZZA が提供できる最上位の答え**:
- 「東京都モス 192 店舗のうち **大半 (推定 150+ 店)** は**株式会社モスストア
  カンパニー**が運営し、残り数十店舗は小規模 FC (ヴィアン含む) による運営」
- 「全国ベースの Mos FC 事業会社は上記 13 社 (法人番号・本社住所・モス外業態・
  出典) が証拠付きで特定済」

これは PI-ZZA の主目的「**メガフランチャイジー (20+店) の特定**」としては十分
機能しており (モスストア / 大和フーヅ / ありがとうサービス / ヴィアン等)、
店舗-operator の 1:1 紐付けは Mos ブランドの情報開示水準の制約で次 phase 対象。
