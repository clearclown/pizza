## 📉 正直な結論

**「東京都内のモスバーガー店舗を運営している事業会社」は公開情報上で構造的に**
**2 社しか確定できない** (モスストアカンパニー + ヴィアン)。

これは PI-ZZA の能力不足ではなく:
- モスフード本部が個別 FC 加盟企業名を非公開としている
- 有報に FC 加盟企業個別名は記載されない (子会社のみ記載)
- BC 誌 202 位までモス掲載はみちのくジャパン (岩手) 1 社のみ
- Google Places API は operator 粒度で検索できない

という **Mos ブランドの情報開示水準そのものの制約**。

追加で特定するには以下のいずれかが必要 (いずれも公開 Web 到達外):
1. 個別店舗の事業所情報を物理的収集 (表札 / 食品衛生許可の営業者 / 求人媒体)
2. 帝国データバンク/東京商工リサーチ有料 DB で産業分類 7721 (ハンバーガー店) × 東京都 フィルタ
3. モス公式 FC 加盟店ポータル (加盟企業限定アクセス)
